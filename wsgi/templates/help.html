    Документация

    Справочная информация по проекту Map Feature Server (mapfeatureserver).

    Проект начат в день 2013-03-12, хотя идея возникла несколько раньше.
    Концепция проекта заключается в том, чтобы, создав специальную серверную программу, дать возможность пользователям разных вьюверов, типа Картобонус (Cartobonus), добавлять в свои карты слои данных (Feature Layer) из любых источников, не только с серверов ArcGIS.
    Для реализации задуманного, mapfeatureserver должен реализовать спецификации http://resources.arcgis.com/en/help/rest/apiref/index.html?featureserver.html, одним из примеров реализации которых является «родной» сервер ArcGIS, к примеру http://sampleserver3.arcgisonline.com/ArcGIS/rest/services/Petroleum/KSPetro/MapServer/1
