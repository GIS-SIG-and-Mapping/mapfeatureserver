    Документация

    Справочная информация по проекту Map Feature Server (Mapfeatureserver).

    Проект начат в день 2013-03-12, хотя идея возникла несколько раньше.
    Концепция проекта заключается в том, чтобы, создав специальную серверную программу, дать возможность пользователям разных вьюверов, типа Картобонус (Cartobonus), добавлять в свои карты слои данных (FeatureLayer) из любых источников, не только с серверов ArcGIS.
    Для реализации задуманного, mapfeatureserver должен реализовать спецификации Feature Service - Layer, одним из примеров реализации которых является «родной» сервер ArcGIS, к примеру http://sampleserver3.arcgisonline.com/ArcGIS/rest/services/Petroleum/KSFields/FeatureServer/0

    Подробные сведения о проекте и его применении можно получить изучив файлы README, входящие в состав проекта.
