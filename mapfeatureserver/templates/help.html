    Документация

    Справочная информация по проекту Map Feature Server (Mapfeatureserver, MFS).

    MFS это свободный (Open Source, GNU license) веб-сервис для замены ArcGIS Server. Уточняю: MFS может заменить ArcGIS Server только в части доступа к данным через ArcGIS Server REST API, подраздел Feature Service.

    Проект начат в день 2013-03-12, хотя идея возникла несколько раньше.
    Концепция проекта заключается в том, чтобы, создав специальную серверную программу, дать возможность пользователям разных вьюверов, умеющих использовать ArcGIS Server REST API, типа Картобонус (Cartobonus), добавлять в свои карты слои данных (FeatureLayer) из любых источников, не только с серверов ArcGIS.
    Для достижения задуманного, mapfeatureserver должен уметь имитировать ArcGIS Server, а именно - обрабатывать запросы и отвечать на них согласно спецификациям ArcGIS Server REST API - Feature Service - Layer. Одним из примеров реализации которых является «родной» сервер ArcGIS, вроде этого: http://sampleserver3.arcgisonline.com/ArcGIS/rest/services/Petroleum/KSFields/FeatureServer/0

    Подробные сведения о проекте и его применении можно получить изучив файлы README, входящие в состав проекта.
